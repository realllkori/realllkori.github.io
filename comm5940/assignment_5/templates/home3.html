Web App 框架

本應用程式建立於 Python Flask framework 和jinja template system 它們共同提供了用戶介面的必要元素。 

數據庫

Airtable數據庫來源於 the Airtable REST API而MySQL數據來源於 Flask SQLAlchemy ORM library.它們提供了佈局數據、分頁和CRUD支持。

Airtable數據來源

這是網站的數據來源。包含狗隻種類、地圖等訊息。通過點擊對應按鈕，您可以嘗試自行加入、改變或刪除數據。
